# Research Collaboration System
## NoSQL Database Project - Palestine Polytechnic University

A comprehensive research collaboration system using multiple NoSQL databases:
- **MongoDB** - Document storage for researchers, projects, publications
- **Neo4j** - Graph database for collaboration networks
- **Redis** - Cache and session management

## Features

### For Researchers
- Profile management
- Project creation and management
- Publication tracking
- Collaboration network visualization
- Search for other researchers

### For Administrators
- System monitoring
- User management
- Analytics and statistics
- Collaboration network analysis
